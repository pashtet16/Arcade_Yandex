# Farming Simulator - Документация

## Обзор игры
**Farming Simulator** — это 2D-симулятор фермы, разработанный на языке Python с использованием библиотеки Arcade. Игрок управляет фермером, сажает культуры, нанимает рабочих и развивает свое хозяйство, чтобы достичь финансовой цели.

## Как запустить
Для запуска игры необходимо:
1. Установить Python (версии 3.10+).
2. Установить библиотеку Arcade: `pip install arcade`.
3. Запустить файл `main.py`: `python main.py`.

## Управление
- **WASD**: Перемещение игрока.
- **Левая кнопка мыши (ЛКМ)**:
    - Сбор урожая (нажать на созревшую культуру).
    - Посадка (нажать на свободную вспаханную ячейку).
    - Взаимодействие с интерфейсом (улучшения, покупка полей).
- **Наведение на рабочего**: Показывает статистику рабочего (уровень, энергия, скорость).

## Геймплейные механики
1. **Посадка и рост**: Каждая культура проходит три стадии: семена, рост, созревание. У разных культур разное время роста и цена продажи.
2. **Рабочие**: 
    - **Сеятели (Planters)**: Автоматически сажают семена на пустые клетки.
    - **Сборщики (Harvesters)**: Автоматически собирают созревший урожай.
    - **Энергия**: Рабочие тратят энергию на действия и уходят на "сон" для восстановления.
3. **Улучшения**: 
    - Можно улучшать скорость передвижения, скорость работы и запас энергии каждого рабочего.
    - Покупка новых полей открывает более дорогие культуры.
4. **Экономика**: Игрок начинает с начальным капиталом и зарабатывает деньги, продавая собранный урожай. Цель игры — заработать $1000.
5. **Сохранение**: Игра автоматически сохраняет прогресс (деньги, открытые поля, уровни рабочих) каждые 5 секунд.

## Архитектура проекта
- `main.py`: Точка входа, инициализация окна.
- `game_view.py`: Основной игровой цикл, обработка физики, событий и камеры.
- `sprites.py`: Классы игровых объектов (`Crop`, `Worker`, `Farmer`).
- `field.py`: Логика управления игровым полем и сеткой.
- `ui_manager.py`: Система интерфейса (HUD, окна улучшений).
- `db_manager.py`: Работа с сохранением данных (SQLite).
- `constants.py`: Глобальные настройки игры.
- `assets/`: Папка с графическими и звуковыми ресурсами.
